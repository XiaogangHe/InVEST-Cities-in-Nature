## List of key input

| Field/Variable                     | Unit                | Value               | Required/Optional   |
| -----------------------------------| ------------------- |---------------------|---------------------|
| Work space                         | -                   | -                   | Required            |
| Area of Interest                   | -                   | -                   | Required            |
| Rainfall Depth                     | mm                  | 100                 | Required            |
| Land Use/ Land Cover               | -                   | -                   | Required            |

| Built Infrastructure               | -                   | -                   | Optional            |
| Damage Loss Table                  | USD $/m²            | -                   | Required            |

Values in this table are **recommended** values. You may need to adjust these values based on your district.

:::
